前言

这仅仅只是一个入门级游戏，一个小toy，使用的也就是贴图形式的pygame，因为需要过多的访存次数，图片的精度也较高，所以对内存以及硬盘的速度有着较高的要求。当时答辩的辣鸡主机上，我们跑的就相当卡。

环境要求

语言版本：python 3.6.2

依赖：pygame

系统：皆可

游戏按键

暂停/开始：Space

退出：Esc
